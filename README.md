# 🚴 Bike Town

**Play now → [biketown.vercel.app](https://biketown.vercel.app)**

Ride your bike around town, collect stars, and explore!

## Controls

| Key | Action |
|-----|--------|
| ⬆️ Arrow / W | Pedal forward |
| ⬇️ Arrow / S | Brake / reverse |
| ⬅️ Arrow / A | Turn left |
| ➡️ Arrow / D | Turn right |

On iPad? Tap the arrow buttons on screen!
